# **App Name**: MX Sweeps Exchange

## Core Features:

- Account Synchronization: Automatically synchronize user account data, including sweeps coin balance, from MXRacehub using the user's MXRacehub credentials.
- Sweeps Coin Loading: Automatically load sweeps coins sent from MXRacehub into the user's exchange account.
- USD Exchange: Allow users to exchange sweeps coins for USD at a fixed rate (100 sweeps coins = $1 USD).
- Transaction History: Maintain a detailed history of all sweeps coin and USD transactions.
- Withdrawal Functionality: Enable users to withdraw USD from their exchange account to an external payment method.
- Compliance Tool: An internal tool that alerts adminstrators when unusual, suspicious, or large transcations take place. This tool will monitor exchange rates and user behavior and other compliance needs.

## Style Guidelines:

- Primary color: Deep purple (#483D8B), a modern, subdued color reminiscent of cryptocurrency aesthetics.
- Background color: Dark slate gray (#2F4F4F), for high contrast against text.
- Accent color: Electric blue (#7DF9FF), used sparingly to highlight interactive elements and important information.
- Font: 'Inter' (sans-serif) for a clean and modern interface. Use for both headers and body text.
- Use simple, geometric icons that align with the app's overall modern aesthetic.
- Maintain a clean, card-based layout to present exchange data in a structured manner.
- Incorporate subtle transitions and animations when displaying balance updates and transaction confirmations.
